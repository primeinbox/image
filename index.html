API Reference
GET

Generate Image — Two Step

# Step 1 — Start generation GET https://aerivue.onrender.com/generate ?prompt=a futuristic lion logo &apikey=img_your_key_here # Response → receive task_id { "task_id": "abc123...", "credit": "@aerivue" } # Step 2 — Poll for result (every 3s) GET https://aerivue.onrender.com/generate/check ?task_id=abc123... &apikey=img_your_key_here # Response when ready { "image_url": "https://...", "credit": "@aerivue" }
ParameterStatusDescription
prompt required Text description of the image you want to generate
apikey required Your img_ prefixed API key from the admin panel
task_id check only Task ID returned from the /generate endpoint
